# Unity Simeple FPS Controller
*Hello everyone! This is my simple **FPS Controller template for Unity**. I use it frequently in my projects and wanted to share it. You are free to use it in your projects (commercial and non-commercial) without attribution.*

**I don't claim that this code is correct or clean. I'm not a C# senior.**

## More about Controller
- The project and controller use free models and animations from [Mixamo](https://www.mixamo.com/).
- The project uses ProBuilder.
- The project uses Cinemachine.
- Currently, the controller provides the following functions:
	- Walking forward
	- Walking backwards
	- Walking left (Left strafe)
	- Walking right (Right strafe)
	- Running forward
	- Controlling the camera with `Cinemachine`.
	- Camera shaking when walking/running.
    - Calculate Stamina.
